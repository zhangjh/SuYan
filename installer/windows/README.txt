IME Pinyin - 跨平台中文拼音输入法
=====================================

版本: 1.0.0
基于: RIME 输入法引擎 + 雾凇拼音词库

功能特性
--------
- 简体中文拼音输入
- 英文直接输入
- 中英文快速切换 (Shift 键)
- 智能词频学习
- 自动学词功能
- 云端词库更新 (计划中)

快捷键
------
- Shift: 切换中英文模式
- Space: 选择第一个候选词
- 1-9: 选择对应候选词
- Enter: 提交拼音原文
- Escape: 清空输入
- Backspace: 删除最后一个字符
- Page Up/Down: 翻页
- +/-: 翻页

文件位置
--------
- 程序目录: C:\Program Files\IMEPinyin\ime-pinyin-x.x.x\
- 用户数据: %APPDATA%\IMEPinyin\

技术支持

- 问题反馈: https://github.com/example/ime-pinyin/issues

=====================================
IME Pinyin - Cross-Platform Chinese Input Method

Version: 1.0.0
Based on: RIME Engine + rime-ice Dictionary

Features
--------
- Simplified Chinese Pinyin input
- Direct English input
- Quick Chinese/English switch (Shift key)
- Smart frequency learning
- Auto word learning
- Cloud dictionary updates (planned)

Keyboard Shortcuts
------------------
- Shift: Toggle Chinese/English mode
- Space: Select first candidate
- 1-9: Select corresponding candidate
- Enter: Commit raw pinyin
- Escape: Clear input
- Backspace: Delete last character
- Page Up/Down: Page navigation
- +/-: Page navigation

File Locations
--------------

- User data: %APPDATA%\IMEPinyin\

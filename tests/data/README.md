# 测试数据

本目录包含用于单元测试和属性测试的测试数据。

## test_dictionary.dict.yaml

测试词库，包含约 1000 条词条，用于验证输入法核心功能。

### 词库内容分类

| 分类 | 词条数 | 说明 |
|------|--------|------|
| 高频单字 | ~100 | 最常用的单字 |
| 常用词组 | ~200 | 日常使用的词组 |
| 动词 | ~80 | 常用动词 |
| 形容词 | ~70 | 常用形容词 |
| 名词 | ~100 | 物品和概念名词 |
| 数字量词 | ~60 | 数字和量词 |
| 网络流行语 | ~80 | 网络用语和日常用语 |
| IT术语 | ~70 | 互联网和编程术语 |
| 方位介词 | ~60 | 方位词和介词 |
| 连词副词 | ~60 | 连词和副词 |
| 疑问代词 | ~50 | 疑问词和代词 |
| 常用短语 | ~50 | 常用短语组合 |
| 模糊音测试 | ~20 | 用于测试模糊音处理 |
| 多音字测试 | ~20 | 用于测试多音字处理 |
| 边界测试 | ~20 | 用于测试边界情况 |

### 词库格式

采用 RIME 标准词库格式 (.dict.yaml)：

```yaml
---
name: test_dictionary
version: "1.0.0"
sort: by_weight
columns:
  - text      # 词条文本
  - code      # 拼音编码
  - weight    # 词频权重
...

中国	zhong guo	97000
```

### 使用方法

在测试中加载此词库：

```cpp
// C++ 示例
DictionaryManager dm;
dm.LoadDictionary("tests/data/test_dictionary.dict.yaml");
```

### 注意事项

1. 此词库仅用于测试，不适合生产环境
2. 词频权重为测试设定，不反映真实使用频率
3. 包含模糊音和多音字测试用例，用于验证边界情况
